# 🛕 Murugan Pilgrim App

A Flutter app built for Murugan devotees to explore the **6 Arupadai Veedu temples** with temple details, live countdowns, darshan timings, booking links, maps, and festival notifications.

---

## 📌 About the Project

Murugan Pilgrim is a spiritual travel guide app designed to help devotees easily access information about Lord Murugan’s sacred temples in Tamil Nadu.

The app provides temple timings, travel guidance, festival reminders, booking links, and maps in one place.

---

## ❓ Problem Statement

Many devotees face difficulties such as:

- Finding correct temple timings  
- Booking darshan tickets online  
- Knowing festival dates  
- Getting route/maps information  
- Searching all 6 Murugan temples in one app  

This app solves these problems with a simple mobile solution.

---

## 🎯 Target Users

- Murugan Devotees  
- Pilgrims  
- Tourists visiting Tamil Nadu temples  
- Elderly users needing easy guidance  
- Families planning temple trips

## 🛕 6 Arupadai Veedu Temples
